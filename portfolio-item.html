Spotify Widget

*Almost* as good as the real thing!

As part of our submission for nwHacks 2022 (a popular hackathon in Western Canada), my three friends and I decided to make a Spotify web app. This widget allows one to log in using their real Spotify account.

The nav bar allows the user to try a variety of different functions. The first one is the "Liked Songs" tab, which displays a handful of the user's liked songs. It displays the song title and artist, as well as the accompanying album cover. If you click on any album cover, it opens the actual Spotify page of that particular album in a new tab.

The next tab, "Currently Playing", can display the song title, artist, and album cover of the song that the user is currently playing, in real time.

The next tab, "Artist Top Tracks", allows the user to search for an artist in the text box, and it will return a list of the top 10 most popular songs and accompanying album covers for those particular songs.

The next tab, "Recommendations", is my personal favourite feature of this entire widget. The user can type in any genre, and a list of song names, artists, and their respective album covers will show up. The primary intention behind this feature is to introduce the user to some new songs that they've never heard before, based on genre. For example, if someone really likes rock music, the resulting list of songs will be all from the rock genre, catering to their personal taste and preference.

The next tab, "Trending", displays a list of the top 50 trending songs globally, in case the user is curious about what kind of music the world is tuned into nowadays.

The last tab, "My Profile", displays the user's profile picture along with their username.

Technologies & Languages Used:

  • Python
  • HTML
  • Flask
  • Jinja
  • CSS
  • Spotipy (Spotify API for Python)
  • Docker
